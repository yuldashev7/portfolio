## 🛠️ Tech Stack & Architecture

This portfolio is engineered using modern, production-ready frontend technologies:

- **Framework:** [Next.js](https://nextjs.org/) (App Router Architecture)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed for reliability)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first styling ecosystem)
- **Deployment:** [Vercel](https://vercel.com/) (Global Edge Network)

### 📐 Architectural Highlights

- **Modular Layouts:** Built with highly reusable UI components ensuring strict DRY (Don't Repeat Yourself) principles.
- **Optimized Performance:** Fully integrated with native Next.js optimization tools to guarantee lightning-fast load times.
- **Glassmorphic Navbar:** Dynamic window-scroll event listener that transitions background styling smoothly using Tailwind's hardware-accelerated transitions.

---

## ✨ Features

- **Responsive UI/UX:** Perfectly adapted for mobile, tablet, and widescreen desktop displays.
- **Interactive Experience Carousel:** A custom interactive layout built to display professional history and stack capabilities elegantly without cluttering the viewport.
- **Performance-First Asset Delivery:** Fully integrated with `next/image` and custom Google Font optimization yielding excellent SEO and sub-second load times.

## 📦 Getting Started & Local Installation

Follow these simple steps to run this project locally on your machine.

### 1. Clone the Repository

```bash
git clone [https://github.com/yuldashev7/portfolio.git](https://github.com/yuldashev7/portfolio.git)
cd portfolio
2. Install Dependencies
Choose the package manager you prefer:

Bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
3. Run the Development Server
Start the local server based on your package manager:

Bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
Open http://localhost:3000 in your browser to view the local instance.
```
